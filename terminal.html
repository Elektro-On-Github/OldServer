Terminale

Benvenuto nel Terminale!!
Per iniziare scrivi un percorso di una directory
presente nel server pubblico. Es: /main/homepage.html
La root del terminale é elektrowindows.ddns.net/
non si puó accedere a directory di sistema (Es: /Windows/System32)
per motivi di sicurezza. Se non conosci il sito web dai uno sguardo alla mappa del server
che trovi anche nella zona degli "smanettoni"
per utilizzare il terminale abilita i popup per la scheda del browser
